# How to use Entity-Component-System to create Match-3 game with TypeScript.

This repository provides game bootstrap for the article:
- [How to use Entity-Component-System to create Match-3 game with TypeScript. Part1](https://medium.com/@rdolivo/ecs-for-gamedev-with-typescript-5a1204f594bc)

### Start with
**Clone** 
```bash
git clone https://github.com/mayakwd/ecs-game-article.git ecs-game
cd ecs-game
```

**Install dependencies**
- Yarn  
    ```bash
    yarn install 
    ```
- NPM  
    ```bash
    npm i 
    ```

### Run
- Yarn
    ```bash
    yarn serve
    ```
- NPM
    ```bash
    npm run serve
    ```
- Open http://localhost:9000/ in the browser.
